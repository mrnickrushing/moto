# MOTO Mayhem Rodeo Facebook archive

Collected from the public MOTO Mayhem Rodeo Facebook profile on July 21, 2026.

Source: <https://www.facebook.com/people/MOTO-Mayhem-Rodeo/61590379212637/>

This folder is an asset archive only. No website source code or existing site data was changed.

## Contents

- `images/original/` — 45 highest-resolution public Facebook photo files recovered from the complete public previous/next photo chain. Facebook's original filenames are preserved.
- `data/photo-posts.json` — source photo IDs, permanent Facebook photo links, publish times, exact page-authored captions, and the matching filename in `images/original/`.
- `videos/` — the public 15-second Facebook reel, 15 one-frame-per-second stills, thumbnail, scrubber sprite, and soundtrack artwork.
- `data/video-source.json` — reel URL and public video metadata.
- `data/media-source.json` — initial public-media discovery metadata and temporary Facebook CDN source URLs. Those CDN URLs expire; the local files are the durable copies.
- `owner-provided/MotoMayhemRodeo_Business_Assets.zip` — the original ZIP supplied by the owner/developer.
- `owner-provided/extracted/` — all 45 human-named images and the README extracted from that ZIP.

Only material authored by the public business profile was retained. Third-party comments, commenter identities, and unrelated profile pictures were excluded.
